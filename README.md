## Функциональность

### Главное окно
- Навигация по трём страницам приложения
- Подтверждение выхода из приложения
- Адаптивный дизайн

### Страница 1
- Ввод значений X, Y, Z
- Проверка области определения (x + y > 0)
- Вычисление сложной математической функции
- Обработка переполнений и ошибок ввода

### Страница 2
- Выбор одной из трёх функций f(x): sh(x), x², eˣ
- Реализация условных вычислений
- Проверка логарифмов (f(x) > 0, |f(x)/y| > 0)

### Страница 3
- Табуляция функции с заданными параметрами
- Построение графика с помощью OxyPlot
- Автомасштабирование осей
- Вывод результатов в многострочное поле

## Тестирование

### Проверка заполнения полей
- Все поля должны быть заполнены
- Корректность ввода чисел
- Использование запятой для дробной части

### Проверка области определения
- **Стр 1:** x + y > 0 (подкоренное выражение)
- **Стр 3:** cos(x) ≠ 0 (тангенс)

### Проверка переполнения
- Обработка Infinity и NaN
- Информативные сообщения об ошибках

## Возможные ошибки и их обработка
  - Пустое поле: "Пожалуйста, заполните поле числом"
  - В поле введено не число: "Поле должно быть числом!"
  - x + y ≤ 0: "Область определения: (x+y) > 0"
  - Огромные числа для вычисления: "Слишком большие значения для вычисления! Пожалуйста, введите меньшие значения"
  - Деление на ноль: "Произошло деление на ноль"
